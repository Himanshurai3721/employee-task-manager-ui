# Employee Task Management System (Frontend)

A simple and clean Employee Task Management UI built using **HTML, CSS, JavaScript, Bootstrap, and jQuery**.

# Features
- Login page
- Dashboard with task statistics
- Add new tasks
- Manage tasks (view & delete)
- Responsive UI
- Integrated with backend API (ASP.NET Core)

# Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- jQuery
